# Dark Slide Layout - Usage Guide

**Date:** January 11, 2025
**Purpose:** Strategic guidance for using dark slides to create visual emphasis and engagement

---

## Overview

Dark slides use dark backgrounds (dark gray, dark purple, navy) with light text to create **visual contrast and emphasis**. They break up lecture monotony and signal important content that deserves special attention.

**Target:** 2-4 dark slides per 22-30 slide lecture (roughly 10-15% of slides)

---

## When NOT to Use Dark Slides

‚ùå **Don't use for:**
- Abstract definitions or theory
- Lists of concepts (use standard or card layout)
- Data tables (use standard or comparison-table)
- Activities (use activity-slide)
- Framework diagrams (use framework-slide)
- Vocabulary or objectives

‚ùå **Don't overuse:**
- Maximum 4 dark slides per lecture
- Distribute throughout (not clustered)
- If you use 5+ dark slides, they lose impact

## Visual Characteristics

**Background:**
- Dark gray (#131313)
- Dark purple
- Navy blue

**Text:**
- Cream/off-white
- Light gray
- White

**Accent Colors:**
- Orange (brand accent)
- Can use images or icons
- Minimal decorative elements

**Effect:**
- High contrast
- Draws immediate attention
- Feels more intimate/serious
- Signals "pay attention to this"

## Tips for Effective Dark Slides

**‚úÖ DO:**
- Use specific, compelling titles
- Keep content focused (100-150 words max)
- Include concrete details (names, numbers, specifics)
- End with clear takeaway or lesson
- Cite sources for case studies

**‚ùå DON'T:**
- Use for abstract theory
- Overload with text
- Cluster multiple dark slides together
- Use without clear purpose
- Skip the "why this matters" connection

---

**Remember:** Dark slides are like spices - use strategically for maximum impact, not liberally throughout!
