## Citation Placement

- **Statistics:** Always cite source
- **Frameworks:** Cite originator (e.g., "Cialdini's 6 Principles (Cialdini, 1984)")
- **Quotes:** Include page number if available
- **Case studies:** Cite source or note "adapted from"

## Anti-Patterns

- Uncited statistics ("73% of managers...")
- Framework without attribution
- Broken or missing URLs
- Inconsistent format within same document

## If URLs Break

- Check Wayback Machine: `web.archive.org/web/[URL]`
- Search author + title on Google Scholar for alternate
- Note in references: "[Archived version]" with archive URL
