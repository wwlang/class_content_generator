# Open Access Sources for Academic Research

A curated list of reliable open-access sources for finding business and management articles that meet our strict accessibility requirements.

---

## Top-Tier Practitioner Publications (Often Open Access)

### Harvard Business Review
**URL:** https://hbr.org
**Access:** Many articles freely accessible
**Topics:** Business strategy, leadership, management, innovation
**Quality:** Top-tier practitioner publication
**Search Tips:**
- Many recent articles are open access
- Use site search: `site:hbr.org [topic]`
- Excellent for applied, practical content

### MIT Sloan Management Review
**URL:** https://sloanreview.mit.edu
**Access:** Selected articles openly accessible
**Topics:** Management research, digital transformation, strategy
**Quality:** Research-backed practitioner insights
**Search Tips:**
- Look for "free article" indicators
- Strong on technology and innovation topics

## Academic Open Access Repositories

### SSRN (Social Science Research Network)
**URL:** https://www.ssrn.com
**Access:** Many working papers freely available
**Topics:** Business, economics, finance, management
**Quality:** Pre-prints and working papers, often from top researchers
**Search Tips:**
- Filter by "Open Access"
- Check author affiliations (top schools)
- Verify if published version is also available

### ResearchGate
**URL:** https://www.researchgate.net
**Access:** Authors share pre-prints and published versions
**Topics:** All academic disciplines
**Quality:** Varies - check original publication venue
**Search Tips:**
- Search for specific authors from top schools
- Request full-text if not immediately available
- Verify publication details

### Academia.edu
**URL:** https://www.academia.edu
**Access:** Author-uploaded papers
**Topics:** All academic disciplines
**Quality:** Varies - verify publication details
**Search Tips:**
- Search by author name
- Check for institutional affiliation
- Verify against published version

### arXiv
**URL:** https://arxiv.org
**Access:** Fully open access pre-print repository
**Topics:** Economics, quantitative finance, statistics
**Quality:** Pre-prints from academic researchers
**Search Tips:**
- Use econ.* categories for economics
- Check if paper was later published
- Good for recent research

## Google Scholar Strategies

### Finding Open Access Versions
**URL:** https://scholar.google.com
**Strategy:**
1. Search for article title or topic
2. Look for [PDF] links on right side
3. Click "All X versions" to find open versions
4. Check for institutional repository versions

### Advanced Search Tips
```
[topic] filetype:pdf site:.edu
[author name] [topic] -paywall
"open access" [topic] [year range]
[concept] "freely available"
```

## Verification Tools

### Check Paywall Status
1. Use WebFetch to load URL
2. Look for these indicators:

**Open Access Indicators:**
- Full text immediately visible
- PDF download button available
- No "Purchase" or "Subscribe" buttons
- "Open Access" badge or icon

**Paywall Indicators:**
- Only abstract visible
- "Purchase article" button
- "Subscribe to access" message
- Login required for full text

### DOI Resolution
**URL:** https://doi.org
- Enter DOI to find official version
- Check if open access version exists
- Look for "green" or "gold" open access indicators

## Fallback Strategies

### When Article is Paywalled

**1. Find Author's Version**
```
Search: [author name] [article title] PDF
Check: Author's university webpage
Check: ResearchGate, Academia.edu
```

**2. Find Pre-Print Version**
```
Search: [article title] working paper
Check: SSRN, arXiv
Check: University repository
```

**3. Find Open Access Alternative**
```
Search: [key concepts] open access
Search: [topic] freely available peer reviewed
Filter: Recent (2020+) open access journals
```

**4. Find Similar Article**
```
Search: [key concepts] [framework] open access
Look: Different author covering same concepts
Validate: New article covers all required concepts
```

## Quality Verification Checklist

When using open-access sources:

**Source Quality:**
- [ ] Is this a peer-reviewed journal?
- [ ] Or is it HBR/MIT Sloan Review level?
- [ ] Is the author affiliated with reputable institution?
- [ ] Is the paper highly cited (for older works)?

**Content Quality:**
- [ ] Does it cover all required concepts?
- [ ] Are definitions and examples provided?
- [ ] Is it current or proven relevant?
- [ ] Is writing quality appropriate for students?

**Accessibility:**
- [ ] Can full text be accessed without login?
- [ ] Is PDF downloadable without subscription?
- [ ] Will students be able to access this?
- [ ] Is URL stable (not temporary link)?

## Recommended Search Workflow

**Step 1: Start with Top-Tier Open Sources**
1. Search HBR: `site:hbr.org [topic]`
2. Search MIT Sloan: `site:sloanreview.mit.edu [topic]`
3. Search top school syllabi for reading lists

**Step 2: Academic Search**
1. Google Scholar with open access filters
2. SSRN for working papers
3. Institutional repositories

**Step 3: Verify and Validate**
1. Check accessibility (no paywall)
2. Verify source quality (peer-reviewed or top-tier)
3. Validate content coverage (all concepts present)

**Step 4: Document**
1. Save full APA citation
2. Note DOI and URL
3. Record access verification date
4. Document why article was selected
