## Assessment Philosophy

This structure emphasizes:

- **Comprehensive Understanding**: Final exam tests breadth and depth
- **Applied Knowledge**: Project demonstrates real-world application
- **Progressive Learning**: Quizzes check understanding throughout
- **Milestone-Based**: Project milestones prevent last-minute work
- **Engagement**: Participation rewards active learning

## Typical Timeline

- **Weeks 1-2**: Foundation, project topic selection
- **Weeks 3-4**: Quiz 1, project proposal due
- **Weeks 5-7**: Quiz 2, continued learning, project work
- **Weeks 8-9**: Quiz 3, project draft due
- **Week 10-11**: Quiz 4, final project due, exam prep
- **Week 12+**: Final exam

## Rubrics Required

1. Project Rubric (specific to project type)
2. Exam Grading Rubric (if using essay/case components)
3. Participation Rubric (optional)
