## Assessment Philosophy

This structure emphasizes:

- **Skill Development**: Portfolio pieces show progressive mastery
- **Collaboration**: Group presentations build teamwork skills
- **Individual Accountability**: Individual marks within group work
- **Knowledge Check**: Quizzes ensure conceptual understanding
- **Engagement**: Participation rewards active learning

## Typical Timeline

- **Weeks 1-3**: Foundation building, first portfolio pieces
- **Weeks 4-6**: Advanced portfolio work, Quiz 1
- **Weeks 7-9**: Final portfolio pieces, Quiz 2, presentation prep
- **Week 10+**: Presentations, Quiz 3

## Rubrics Required

1. Written Work Rubric (adapted per portfolio piece)
2. Group Presentation Rubric
3. Individual Contribution Rubric
4. Participation Rubric (optional)
